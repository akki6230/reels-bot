Privacy Policy — Cosmos Capsule

Last updated: June 2026

What we collect

Cosmos Capsule is an automated educational content bot that posts daily facts about space, history, geography, and science to Instagram. We do not collect any personal data from users.

Instagram data

This app uses the Instagram Graph API solely to publish content to our own Instagram account (@cosmos.capsule). We do not access, store, or process any other user's data.

Third party services

We use the following services: Anthropic Claude API (content generation), Pexels API (images), Freesound API (music), Instagram Graph API (publishing).
